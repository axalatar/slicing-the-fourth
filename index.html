It lets you rotate a position around the origin counterclockwise by some angle on some axis. To use it, we just need to take the three matrices: R_x = [1, 0, 0 \ 0, cos(theta), -sin(theta) \ 0, sin(theta), cos(theta)] R_y = [cos(theta), 0, sin(theta) \ 0, 1, 0 \ -sin(theta), 0, cos(theta)] R_z = [cos(theta), -sin(theta), 0 \ sin(theta), cos(theta), 0 \ 0, 0, 1] and multiply our coordinates [x \ y \ z] by them.
